# LLM Course Assignments

This repository contains notebooks and supporting files for the LLM course assignments.

## Contents

- Assignment 1: introductory coding tasks and notebook work
- Assignment 2: multiple question-based notebook exercises
- Assignment 3: embedding and inversion-related tasks, including auxiliary data files

## Structure

- Assignment1/: notebook for Assignment 1
- Assignment2/: notebooks for Assignment 2 questions 1–3
- Assignment3/: notebooks and data files for Assignment 3
